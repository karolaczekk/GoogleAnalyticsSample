# SQL Data Model

Base View
`base_sessions` filters the dataset to the last 180 days.

Date range logic
Last 180 days are calculated dynamicallybased on the maximum available date in the dataset.

Dimension Tables
These tables enable global filters in Power BI.
- dim_date - calendar table for the last 180 days
- dim_device - device categories
- dim_country - list of countries from dataset
- dim_channel - Google Analytics marketing channel grouping

# Metric Definitions

The following metrics are used in the analysis. All metrics are calculated at the session level.
Core metrics are calculated in BigQuery.
Business KPIs are calculated in Power BI using DAX.

* Sessions - The number of website sessions initiated by users.

`SUM(totals.visits)`

* Users - The number of unique users who visited the website.

`COUNT(DISTINCT fullVisitorId)`

* New Users - Users who visited the website for the first time.

`SUM(totals.newVisits)`

* Pageviews - Total number of pages viewed during all sessions.

`SUM(totals.pageviews)`

* Time on Site - Total time users spent on the website during sessions (in seconds).

`SUM(totals.timeOnSite)`

* Average Pages per Session - Average number of pages viewed during a session.

`Pageviews / Sessions`

* Average Session Duration - Average time users spend during a single session (in seconds).

`Time on Site / Sessions`

* Average Sessions Per User - Average number of sessions generated by a single user.

`Sessions / Users`
